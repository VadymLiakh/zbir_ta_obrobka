# Обробка пропущених значень у енергетичних даних

## Опис завдання
Метою роботи є розробка функцій для обробки пропущених значень у наборі енергетичних даних із використанням різних підходів.

Було реалізовано такі методи:
- імпутація середнім значенням;
- імпутація медіаною;
- заповнення попереднім значенням (forward fill);
- заповнення наступним значенням (backward fill).

---

## Використані технології
- Python 3
- Pandas
- NumPy

---

## Опис реалізації

### 1. Генерація даних
Було створено невеликий набір енергетичних даних, який містить:
- часові мітки (`timestamp`);
- споживання електроенергії (`consumption_kwh`);
- напругу (`voltage`);
- температуру (`temperature`).

---

### 2. Створення пропущених значень
У даних випадковим чином було створено пропуски у різних стовпцях:
- `consumption_kwh`
- `voltage`
- `temperature`

Це дозволило змоделювати реальні ситуації втрати даних.

---

### 3. Реалізовані методи обробки

#### Імпутація середнім значенням
Пропущені значення замінюються середнім значенням стовпця.

#### Імпутація медіаною
Пропущені значення замінюються медіаною стовпця.

#### Заповнення попереднім значенням (ffill)
Кожне пропущене значення замінюється попереднім відомим значенням.

#### Заповнення наступним значенням (bfill)
Кожне пропущене значення замінюється наступним відомим значенням.

---

### 4. Результати
Для прикладу було оброблено стовпець `consumption_kwh`.

Усі методи успішно заповнили пропущені значення:

| Метод                  | Кількість пропусків після |
|------------------------|---------------------------|
| Середнє                | 0                         |
| Медіана                | 0                         |
| Попереднє значення     | 0                         |
| Наступне значення      | 0                         |

---

## Особливості методів
- **Середнє**: добре підходить для рівномірних даних, але чутливе до викидів.
- **Медіана**: більш стійка до викидів.
- **ffill**: добре працює для часових рядів.
- **bfill**: корисне, якщо важливі майбутні значення.

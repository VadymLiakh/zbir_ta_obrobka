# Практична робота №2 (Варіант 10)

## Опис
У цій роботі виконано аналіз даних замовлень за допомогою Python (Pandas, NumPy).  
Дані було згенеровано штучно з урахуванням реалістичних характеристик (випадкові значення, пропуски, шум, дублікати).

## Структура проєкту
- `generate_orders_csv.py` — генерація вхідного датасету `orders.csv`
- `variant_10_analysis.py` — реалізація варіанту 10
- `orders.csv` — згенеровані дані
- `variant_10_result.csv` — результат обробки

## Суть завдання (Варіант 10)
Виконати групування даних за товаром (`product`) та обчислити:
- кількість замовлень (`count`)
- сумарну виручку (`sum`)
- середнє значення (`mean`)
- медіану (`median`)
- стандартне відхилення (`std`)

Дані відсортовано за сумарною виручкою та сформовано підсумкову таблицю:
**"топ-5 товарів + інші"** з обчисленням частки у загальній виручці.

## Основні етапи
1. Завантаження та очищення даних
2. Групування за `product`
3. Обчислення статистичних показників
4. Сортування за виручкою
5. Формування топ-5 товарів
6. Об’єднання інших товарів у категорію "Інші"
7. Обчислення частки у загальній виручці

## Результат
Отримано агреговану таблицю, яка дозволяє визначити:
- найбільш прибуткові товари
- їх вклад у загальну виручку
- статистичні характеристики продажів

Результат збережено у файл `variant_10_result.csv`.

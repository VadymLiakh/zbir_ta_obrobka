# Порівняння методів інтерполяції у часових рядах енергоспоживання

## Опис завдання
Метою роботи є заповнення пропущених значень у часовому ряді енергоспоживання за допомогою різних методів інтерполяції та порівняння їх точності.

У роботі були використані такі методи:
- лінійна інтерполяція;
- поліноміальна інтерполяція;
- сплайн-інтерполяція.

Для оцінки якості відновлення пропущених значень були обчислені метрики похибки:
- **MAE** (середня абсолютна похибка);
- **RMSE** (середньоквадратична похибка).

---

## Опис реалізації

### 1. Генерація часового ряду
У програмі було створено штучний часовий ряд енергоспоживання, який включає:
- загальний тренд;
- сезонну складову;
- випадковий шум.

Це дозволило змоделювати дані, наближені до реального енергоспоживання.

---

### 2. Створення пропущених значень
У часовому ряді випадковим чином було видалено **10% значень**.  
Збережено:
- індекси пропущених значень;
- їх справжні значення для подальшого порівняння.

---

### 3. Використані методи інтерполяції
Для відновлення пропущених значень було застосовано три підходи:

#### Лінійна інтерполяція
Заповнює пропуски прямою лінією між сусідніми точками.

#### Поліноміальна інтерполяція
Використовує поліном другого порядку для апроксимації значень.

#### Сплайн-інтерполяція
Використовує кубічний сплайн для плавного відновлення даних.

---

### 4. Оцінка похибки
Після відновлення значень було порівняно відновлені точки з оригінальними значеннями, які були приховані раніше.

Було отримано такі результати:

| Метод                     | MAE   | RMSE  |
|--------------------------|-------|-------|
| Лінійна інтерполяція     | 2.8043 | 3.3609 |
| Поліноміальна інтерполяція | 2.9455 | 3.8593 |
| Сплайн-інтерполяція      | 3.0987 | 3.7790 |

---

## Результат
За отриманими результатами **найменшу похибку показала лінійна інтерполяція**.

Це означає, що для даного часового ряду саме лінійний метод виявився найбільш точним серед розглянутих.

# Оптимізація використання пам’яті Pandas DataFrame

## Опис завдання
Метою роботи є оптимізація використання пам’яті при обробці великих енергетичних наборів даних у Python за допомогою бібліотеки Pandas.

Завдання включає:
- генерацію великого набору енергетичних даних;
- оптимізацію типів даних (`astype`, `downcast`);
- порівняння використання пам’яті до та після оптимізації.

---

## Використані технології
- Python 3
- Pandas
- NumPy

---

## Опис реалізації

### 1. Генерація даних
Було створено штучний набір даних розміром **1 000 000 рядків**, який містить:
- ідентифікатори лічильників
- регіони
- типи енергії
- споживання (kWh)
- напругу, струм, температуру
- статус системи

---

### 2. Оптимізація типів даних
Було виконано:
- зменшення типів цілих чисел (`int32 → int16`)
- зменшення типів чисел з плаваючою крапкою (`float64 → float32`)
- перетворення текстових значень (`object → category`)
- оптимізація булевих значень

---

### 3. Порівняння пам’яті

| Стан            | Використання пам’яті |
|-----------------|---------------------|
| До оптимізації  | ~190.80 МБ          |
| Після оптимізації | ~20.98 МБ          |

**Економія пам’яті:** ~89%

## Висновок
У результаті роботи було показано, що оптимізація типів даних у Pandas дозволяє суттєво зменшити використання оперативної пам’яті. Це особливо важливо при роботі з великими наборами енергетичних даних.
